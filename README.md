# 🧮 Smart Calculator

A menu-driven Smart Calculator built using Python.

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Power
- Square Root
- Factorial
- Error handling for division by zero
- Continuous execution until Exit

## 🛠 Technologies Used

- Python 3

## 🚀 How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

```bash
python calculator.py
```

## 📚 Concepts Used

- Functions
- Loops
- Conditional Statements
- Exception Handling
- User Input
